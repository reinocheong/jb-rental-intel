# USER.md — Smart Tenancy Pro 用户画像与业务价值

## 🎭 核心用户画像

### 1. 马来西亚房产 Agent (Property Agent)
- **核心痛点：** 需要寻找 cobroke 合作机会，但在 Facebook 群组中海量搜索帖子效率极低，且难以追踪谁曾发过哪些楼盘。
- **目标：** 快速定位拥有特定楼盘房源的同行（Agent），通过 WhatsApp 直接发起合作洽谈。

## 💰 业务价值闭环

1. **数据发现：** 自动化爬虫 24/7 监控 FB 房产群组。
2. **结构化提取：** 将混乱的帖子转化为清晰的「楼盘-电话-租金」清单。
3. **精准触达：** WhatsApp 自动化推广引擎向潜在客户发送合作邀请。
4. **低门槛转化：** 3 天免费试用，扫码/Google 登录即可查看完整数据。
5. **付费订阅：** RM 9.90/月（早鸟价）自动续费，确保持续获取最新房源情报。

## 🗺️ 全局交互潜规则

- **30 天冷却：** 为避免骚扰，同一 Agent 每 30 天只会收到一次推广消息。
- **分时段推广：** 每天分 5 个时段（10:30-14:30）发送，降低 WhatsApp 封号风险。
- **A/B 文案交替：** 自动切换「同行共鸣型」与「价值优先型」文案，测试转化率。
- **静默同步：** 隧道 URL 与房源 JSON 每 30 分钟后台自动更新，用户无感知。
